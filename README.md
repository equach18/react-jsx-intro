# react-jsx-intro
This assignment includes building and rendering React components.

## Part 1: Basic Components
Includes FirstComponent, NamedComponent, and App components.

## Part 2: Tweet Component
Tweet component takes in the props of the username, name, date, and message being tweeted. App component renders three tweets.

## Part 3: Person Component
Person component has name, age, and hobbies properties. A voting message will be displayed given the age. If the person's name is longer than 8 characters, only the first six character of their name will show. The hobbies property accepts and array, which will be listed as an li. App renders three copies of the Person component on the page.
